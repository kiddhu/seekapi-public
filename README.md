# seekapi-public

Public-facing content repository for SeekAPI.

## Purpose

This repository is the public shell for intentionally public assets, including:

- blog content
- SEO / AIO content
- public README material
- SDK and tool-facing public assets

## Boundary

This repository must not contain:

- internal governance documents
- internal workflows
- private project implementation details
- credentials, secrets, or operational recovery details
